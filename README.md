# E-commerce Backend

Backend application built using Spring Boot.

## Overview
This project implements a RESTful backend for an e-commerce system, handling core functionalities such as data management and API processing.

## Tech Stack
- Java
- Spring Boot
- Maven

## Getting Started

### Prerequisites
- Java 17+
- Maven

### Run the Project
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run the main application class
